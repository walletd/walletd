# WalletD Prasaga Avio Integration

## Setup
1. Build: `cargo build --release`
2. Run CLI: `cargo run --bin walletd-icp-cli`
3. Select option 9 for Prasaga Avio

## Features
- Address generation
- Balance checking
- SAGA token transfers
- Network status monitoring
